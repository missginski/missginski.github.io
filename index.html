Drop Shop

A fun take on a weather app, that uses the daily forecast to curate outfit suggestions. I wanted to play with the idea of tying together different pieces of, more or less, unrelated data. I built a database to act as a go between for a weather and a shopping API. For this project, I also included a user table with password encryption.

Built Using

  • Express
  • PG-Promise
  • Bcrypt
  • Open Weather Map
  • Shopstyle Collective

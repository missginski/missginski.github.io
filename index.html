Drop Shop

Summary

A fun take on a weather app, that uses the forecast to help curate an outfit for the day. I wanted to experiment with the idea of tying together different pieces of, more or less, unrelated data. I built a database to act as a go between for a weather and a shopping API. For this project, I also included a user table with password encryption.

Built Using

  • Express
  • PG-Promise
  • Bcrypt
  • Open Weather Map
  • Shopstyle Collective
